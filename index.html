Z-index

The z-index property specifies the stack order of an element.
W3C Wiki

Values

  • auto

    The stack level of the generated box in the current stacking context is '0'. The box does not establish a new stacking context unless it is the root element.

  • <integer>

    This integer is the stack level of the generated box in the current stacking context. The box also establishes a new stacking context.

  • inherit

    Takes the same specified value as the property for the element's parent.

See the Pen jVdzoX by vend3tta (@vend3tta) on CodePen.

Notes

  • Element with [positive] z-index always will be above element without it.
  • Lower amount of z-index elements is better.
  • On the same level the last element will be the highest one.
  • In most cases you actually do not need z-index.
